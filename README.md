# Hig Landing Page

A simple static landing page for Hig.

## Edit before publishing

1. Replace `yourdomain.com` inside `CNAME` with your real domain.
2. Replace the App Store and Google Play `href="#"` links in `index.html` with your real app links.
3. Replace `Privacy` and `Terms` links in `index.html` with your real policy URLs.

## GitHub Pages

Upload all files in this folder to a GitHub repository, then enable GitHub Pages from Settings → Pages.
